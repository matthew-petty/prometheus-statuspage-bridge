# Contributing

Thanks for your interest in contributing!

## Roadmap

The next planned enhancement is improving the incident message templating system.

## Expectations

This project is maintained as a low priority for our organization. We welcome pull requests but response times may be slow. If you need a feature urgently, consider forking.

## Submitting Changes

1. Fork the repo
2. Create a branch
3. Ensure tests pass: `pytest tests/ -v`
4. Open a pull request
